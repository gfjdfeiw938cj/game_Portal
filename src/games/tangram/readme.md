Структура:

tangram.html          # Основной файл игры
tangram.js            # Логика перетаскивания фигур
tangram-pieces.js     # Создание SVG-элементов танграма
hint.js               # Данные и функции подсказок
solution-checker.js   # Проверка правильности сборки
mode-manager.js       # Переключение режимов игры/создания
debug-tools.js        # Инструменты создания новых фигур

reset.css             # Личный ресет для страницы танграм, в идеале интегрировать, нет времени
styles.css            # Стили интерфейса

Про SVG фигуры: 
Создается SVG-элемент для каждой фигуры в tangram-pieces. 
Каждая фигура создается в (0;0) а затем через transform перемещается в начальное положение. 
Для перетаскивания используется tangram.js (translate для перетаскивания, rotate для поворота), 
причем изменения отслеживаем в абсолютных координатах ViewBox, а не в пикселях. 
Пермещения объктов ограничено игровым полем. 
Поворот работает на 45 градусов в относительных координатах (0;0) системы коррдинат фигуры.
hint.js хранит образы фигур, которые нужно сложить.
Фигуры одного размера будут засчитаны как корректные и в прямых, и в обратных позициях. 

Плюсы: фигуры ничего не весят, меняются через CSS, масштабируются под любой экран.
Минусы: гемор в расчетах всего и вся начиная с создания самих фигур.
